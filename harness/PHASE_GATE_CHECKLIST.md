# Phase Gate Checklist

- [ ] Goal is clear.
- [ ] Scope and non-goals are documented.
- [ ] Relevant context and logs were read.
- [ ] Files touched are listed.
- [ ] Acceptance criteria are met.
- [ ] Tests or manual checks were run.
- [ ] Results are recorded.
- [ ] Implementation log is updated.
- [ ] Known risks are recorded.
- [ ] Repo hygiene is clean.
- [ ] No `git add .` was used.
