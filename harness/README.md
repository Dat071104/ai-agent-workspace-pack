# Harness

General-purpose checklists and matrices for phase gates, audits, bug triage, consistency checks, production readiness, prompt harness design, and token/risk classification.

These files are stack-neutral and can be copied into target projects or referenced by prompts.

Vietnamese note: Harness la bo checklist/matrix de kiem tra chat luong, gate theo giai doan, va muc rui ro/token.

## Files

- `TOKEN_RISK_MATRIX.md`: classify tasks as Light, Medium, Heavy, or Very Heavy before starting.
- `PHASE_GATE_CHECKLIST.md`: check whether a phase is complete.
- `AUDIT_CHECKLIST.md`: general audit coverage.
- `BUG_TRIAGE_CHECKLIST.md`: verify and classify bugs before fixing.
- `CONSISTENCY_MATRIX.md`: compare completed parts/phases.
- `PRODUCTION_READINESS_MATRIX.md`: release readiness view.
- `PROMPT_HARNESS_TEMPLATE.md`: structure for prompt/tool/test harnesses.

Use `TOKEN_RISK_MATRIX.md` before full audits, production readiness work, clean-code, or multi-phase autonomous prompts.
