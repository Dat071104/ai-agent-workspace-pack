# Bug Triage Checklist

- [ ] Bug report is understood.
- [ ] Expected behavior is identified.
- [ ] Actual behavior is identified.
- [ ] Reproduction steps are attempted or inspected.
- [ ] Context/logs are read.
- [ ] Issue is classified.
- [ ] Affected zone is mapped.
- [ ] Adjacent regression risks are listed.
- [ ] Minimal fix is proposed.
- [ ] Tests are identified.
- [ ] User confirmation is requested before patching.
