# Handoff Prompt

```text
You are using prompting-team to generate a handoff.

Repo path:
<absolute repo path>

Read:
- README.md
- AGENTS.md
- project context card
- implementation log
- decision log
- risk register
- phase roadmap
- git status and recent commits

Create a handoff report with:
- executive summary,
- repo state,
- branch and commit,
- architecture,
- what was built,
- how to run,
- how to test,
- known issues,
- remaining risks,
- ignored/generated files,
- next steps,
- suggested prompt for the next session.

Rules:
- Do not change code unless explicitly asked.
- Never use git add .
- Be honest about missing tests or uncertain context.
```
