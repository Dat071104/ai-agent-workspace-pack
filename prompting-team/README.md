# Prompting Team

Use `prompting-team/` to create strong execution prompts, audit prompts, handoff prompts, production-readiness prompts, and harness prompts.

Prompts from this team should include goal, scope, repo path, initial inspection, context/log reading, phase gates, tests, audits, repo hygiene, implementation log updates, commit/push rules, final report format, and failure honesty.

Vietnamese note: Team nay tao prompt manh, co gate, test, hygiene va bao cao that.

Files:

- `SKILL.md`
- `PROMPT_READINESS_GATE.md`
- `PHASE_BY_PHASE_PROMPT.md`
- `AUDIT_PROMPT.md`
- `HARNESS_PROMPT.md`
- `HANDOFF_PROMPT.md`
- `PROMPT_REVIEW_CHECKLIST.md`

Use `PROMPT_READINESS_GATE.md` before creating large multi-phase, autonomous, production-readiness, or cross-agent prompts. Show a prompt preview in chat and ask before writing prompt files.
