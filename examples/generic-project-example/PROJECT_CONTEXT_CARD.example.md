# Project Context Card Example

## Project Name

Project Alpha Support Desk

## Summary

A local-first web app for a small support team to track customer requests, assign owners, and export weekly summaries.

## Current Phase

Phase 2: ticket list, filtering, and export workflow.

## Tech Stack

- TypeScript
- React
- Local JSON storage
- Playwright smoke tests

## How to Run

```bash
npm install
npm run dev
```

## How to Test

```bash
npm test
```

## Known Risks

- Export format is not final.
- Offline conflict behavior is not implemented.

## Next Step

Add scoped audit for filtering and export.
