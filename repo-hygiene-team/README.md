# Repo Hygiene Team

Use `repo-hygiene-team/` to keep a repository clean, safe, and public-friendly.

It covers:

- `.gitignore`,
- forbidden tracked files,
- generated artifacts,
- secrets,
- local logs,
- public repo naturalness,
- release tags,
- commit safety.

Vietnamese note: Team nay giup repo sach truoc khi commit, publish hoac release.

Files:

- `SKILL.md`
- `GIT_SAFETY_RULES.md`
- `PUBLIC_REPO_HYGIENE_CHECKLIST.md`
- `RELEASE_CHECKLIST.md`
