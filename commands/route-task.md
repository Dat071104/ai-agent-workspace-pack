# Route Task Command

```text
Use TEAM_ROUTER.md to route this request.

Request:
<paste request>

Output:
- Recommended team.
- Why this team fits.
- Token/risk level: Light, Medium, Heavy, or Very Heavy.
- Needed context.
- Expected output.
- Optional skill/team suggestions.
- Confirmation question before loading long team files or writing files.

Rules:
- Do not read every team folder.
- Do not write files.
- Never use git add .
- Warn before expensive or risky work.
```
