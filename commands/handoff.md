# Handoff Command

```text
Use handoff-team.
Read this project's own README.md (not the pack's), AGENTS.md, project context, implementation log, decision log, risk register, roadmap, git status, and recent commit if available.
Create a handoff report with executive summary, repo state, architecture, what was built, how to run, how to test, known issues, next steps, ignored files, and a suggested next-session prompt.
Safety: never use git add ., do not modify code unless asked.
Expected output: complete handoff report.
```
