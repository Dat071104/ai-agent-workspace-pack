# Advise Command

```text
Use advisor-team. Act as a senior advisor/mentor over the whole project.
Read the project context card and idea; read implementation log and risk register if present.
Inspect only representative code. Keep scope Medium.
Assess current state, then rank improvement opportunities by business value with a difficulty score (S/M/L/XL).
Recommend the top 1-3 and name which team should execute each. Say what is not worth doing now.
Safety: read-only, never use git add ., do not change code, mark estimates as known/estimate/needs data.
Expected output: goal restatement, state assessment, opportunity register, difficulty scores, top picks, deferred list, confirmation request.
```
