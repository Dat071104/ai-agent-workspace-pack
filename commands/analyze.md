# Analyze Command

```text
Use analyze-team.
Read this project's own README.md (not the pack's), AGENTS.md, and available project context first.
Clarify the request, create 2-4 options, compare pros/cons/risks, recommend one option, and wait for my confirmation before creating major structure or files.
Safety: never use git add ., ask before destructive actions, and produce an honest report.
Expected output: assumptions, questions, options table, recommendation, confirmation request.
```
