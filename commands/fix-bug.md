# Fix Bug Command

```text
Use bug-fix-team.
Bug report: <paste bug report>.
Read context and logs, reproduce or inspect, classify the report, map the affected zone, propose the minimal fix, and wait for my confirmation before editing.
Safety: never use git add ., ask before destructive actions, commit only if I explicitly allow it.
Expected output before editing: classification, evidence, affected zone, impact analysis, fix proposal, confirmation request.
```
